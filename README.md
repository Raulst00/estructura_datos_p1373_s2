# estructura_datos_p1373_s2# Estructura de Datos - Proyecto Búsquedas
**Repositorio:** estructura_datos_p1349_s2

## Descripción del Proyecto
Este proyecto implementa y compara el rendimiento y comportamiento de dos algoritmos fundamentales de búsqueda: **Búsqueda Secuencial (Lineal)** y **Búsqueda Binaria**. Ambos métodos se evalúan utilizando un arreglo masivo de 1,000,000 de elementos estructurados de manera lineal y ordenada.

## Pre-requisitos
* **Lenguaje:** Python 3.x instalado en el sistema.
* **IDE Recomendado:** Visual Studio Code.
* No se requieren librerías externas (se utilizan los módulos nativos `time`).

## Instrucciones para Ejecutar el Programa
1. Descarga o clona este repositorio.
2. Abre la carpeta del proyecto en Visual Studio Code.
3. Abre una terminal integrada y ejecuta el archivo principal con el comando:
   ```bash
